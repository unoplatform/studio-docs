# Studio Live Overview

[!INCLUDE [preview banner](includes/sl-important-info.md)]

Studio Live is a browser-based development environment for building [Uno Platform](https://platform.uno) applications with AI assistance. It lets you describe what you want to build, iterate on the result in real time, and export a working Uno Platform solution — all without installing anything locally.

## Who Is It For

Studio Live is designed for:

- **Developers** who want to scaffold Uno Platform applications quickly and iterate with AI help.
- **Designers and product owners** who want to prototype ideas and hand off working code.
- **Teams** exploring Uno Platform without committing to a local toolchain setup.

## Core Capabilities

| Capability | Description |
|---|---|
| AI Conversation | Describe features in natural language; the AI agent writes, edits, and explains code. |
| Live Preview | See your application running in the browser as changes are applied. |
| Catalog & Samples | Start from curated, production-ready samples across a range of use cases. |
| Hot Reload | Apply XAML and C# changes instantly without restarting the preview. |
| Export | Download a fully formed Uno Platform solution to continue locally. |
